Annotations sémantiques de pages HTML

Pour cet exercice, vous allez annoter une page HTML : la page personnelle de James Bond.

Cette page traite de sujets différents, vous aurez donc besoin de vocabulaires différents.

Exercice 1

Utilisez le vocabulaire FOAF, ainsi que les propriétés dbo:spouse et dbo:child (voir cette page pour la signification du préfixe dbo) pour annoter en RDFa la biographie.

Vérifiez ces annotations avec le Validateur RDFa.

Exercice 2

Annotez les films en utilisant le marqueur de Google.

Vérifiez la compréhension de ces balises par Google avec cet outil de test. Pour éviter les messages d'erreurs, vous devrez probablement compléter les annotations de l'exercice 1.

Exercice 3

Utilisez le vocabulaire de schema.org/Recipe, pour annoter les recettes de cuisine
